# Instructions for Code-Modifying Agents

Before modifying this repository, read and follow:

- `.policy/ARTIFACT_FRESHNESS_POLICY.md`
- `.policy/artifacts.json`

The artifact freshness policy applies to every human, LLM, automation,
dependency bot, script, local Git hook, and runtime provisioning path that
modifies or selects artifacts for this project.

No artifact may be added, updated, embedded, locked, referenced, distributed,
vendored, generated from a third-party source, or provisioned unless it is the
newest eligible version after the 7-day quarantine period.

There is no exception, override, bypass, emergency exemption, or force mode.
